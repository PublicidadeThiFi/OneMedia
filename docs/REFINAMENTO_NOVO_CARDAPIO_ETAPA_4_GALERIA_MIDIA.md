# Refinamento — Etapa 4 — Galeria de mídia

Fonte verdade das etapas: `Refinamento_Novo_Cardapio_Etapas.pdf`.
Referência visual: `Mídia Kit.pdf`.

## Objetivo
Preparar o frontend para consumir a galeria pública estruturada do cardápio sem quebrar a compatibilidade com os campos antigos de imagem e vídeo.

## Entregas desta etapa
- tipagem pública para itens de mídia (`image` / `video`)
- suporte a `mediaGallery` e `orderedMediaCollection` em pontos e faces/telas
- helpers para derivar galeria do card com fallback seguro
- preservação de `mainImageUrl`, `mainVideoUrl`, `imageUrl`, `videoUrl`, `galleryImages` e `galleryVideos`

## Observação
A aplicação visual do carrossel ainda fica para a etapa específica de interação. Nesta etapa o foco é deixar o contrato pronto para consumo.
